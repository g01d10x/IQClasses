Simple CGGeometry extension C functions to simplify your Geometry calculations. https://github.com/hackiftekhar/IQClasses/tree/master/CGGeometry%2BExtension

CountryStateDemo

This is a demo project. CountryStateHandler iOS classes used to handle Country List and State list according to Country. https://github.com/hackiftekhar/IQClasses/tree/master/CountryStateDemo

Keyboard TextField Manager

Use this classes to handle UITextField and iOS UIKeyboard. so your textfield will not be hide by Keyboard https://github.com/hackiftekhar/IQClasses/tree/master/KeyboardTextFieldDemo

CoreData Helper

Demo project contains coreData helper classes. Use CoreDataHelper to handle all the CoreData related query. Just drag and drop in your project and use it as you want. https://github.com/hackiftekhar/IQClasses/tree/master/CoreDataDemo
